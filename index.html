Privacy Policy – Rozsetu

Rozsetu respects the privacy of its users. This Privacy Policy explains how we collect, use, and protect user information.

Information We Collect

  • Phone number or email for authentication
  • User ID generated by the app
  • Task-related submissions such as screenshots or UPI ID
  • Basic device and app usage information

How We Use Information

  • To verify user identity
  • To process task submissions
  • To review and approve tasks
  • To provide reward payouts
  • To improve app functionality and user support

Data Sharing

We do not sell or share personal user data with third parties except when required for task verification or legal compliance.

Data Security

We use Firebase services and industry-standard security practices to protect user data.

Data Retention and Deletion

User data is retained only as long as necessary. Users can request data deletion by contacting us at support@rozsetu.com.

Children’s Privacy

Rozsetu is intended for users aged 18 years and above.
